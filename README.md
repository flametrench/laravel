# Flametrench Laravel SDK

Laravel SDK for [Flametrench](https://github.com/flametrench/spec).

## Status

Pre-release. The Laravel SDK is not yet implemented. The specification is being drafted at [flametrench/spec](https://github.com/flametrench/spec).

When ready, the SDK will target Laravel 11+ on PHP 8.3+ and will provide identity, tenancy, and authorization primitives that conform to the Flametrench specification.

Related: [`flametrench/ids`](https://github.com/flametrench/ids-php) — the first published Flametrench PHP package.
